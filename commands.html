Cherry Bot Commands

Moderation

/mod ban <user> [reason]

Ban a user from the server

Example: /mod ban @User Breaking rules

/mod kick <user> [reason]

Kick a user from the server

Example: /mod kick @User Spamming

/mod mute <user> <duration> [reason]

Temporarily mute a user

Example: /mod mute @User 1h Excessive caps

/mod clear <amount> [user]

Clear messages in a channel

Example: /mod clear 50

/mod slowmode <seconds>

Set channel slowmode

Example: /mod slowmode 10

Security

/security antiraid <on/off>

Toggle anti-raid protection (default: 6 joins/10s)

/security antinuke <on/off>

Toggle anti-nuke protection (default: 3 deletions/30s)

/security threshold raid <joins> <seconds>

Set raid detection threshold

/security threshold nuke <actions> <seconds>

Set nuke detection threshold

/security logs <channel>

Set security log channel

Music

/music play <song>

Play a song or add to queue

Example: /music play Never Gonna Give You Up

/music pause

Pause current song

/music resume

Resume paused song

/music skip

Skip current song

/music queue

Show current queue

/music volume <0-200>

Set music volume

/music remove <position>

Remove song from queue

/music clear

Clear the queue

Utility

/utility weather <city>

Get weather information

/utility translate <text> <language>

Translate text to another language

/utility calc <expression>

Calculate mathematical expression

/utility reminder <time> <message>

Set a reminder

/utility qr <text>

Generate QR code

/utility shorten <url>

Shorten a URL

Games

/games rps <choice>

Play Rock, Paper, Scissors

/games coinflip

Flip a coin

/games dice [sides]

Roll a dice (default: 6 sides)

/games 8ball <question>

Ask the magic 8-ball

/games random <min> <max>

Generate random number

/games trivia

Play a trivia game

Leveling

/level rank [user]

Check your or another user's rank

/level leaderboard

Show server leaderboard

/level set <user> <level>

Set user's level (admin only)

/level role add <role> <level>

Add level-up role reward

/level role remove <role>

Remove level-up role

/level multiplier <role> <value>

Set XP multiplier for role

AI Chat

/chat <message>

Chat with Cherry's AI

Example: /chat Tell me a story

/chat clear

Clear chat history

/chat style <personality>

Change AI chat style

Example: /chat style mysterious
